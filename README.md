# ⚽ Football Shirts Gallery

Football Shirts Gallery is a small web project where I created a football shirts catalog using HTML, CSS, JavaScript, Bootstrap and JSON Server.

The idea of the project is to load football shirts dynamically from a fake API and display them in a modern responsive gallery with filters and animations.

---

## 🚀 Features

- Dynamic cards generated with JavaScript
- Data loaded from JSON Server using fetch
- Async/await implementation
- Responsive design with Bootstrap
- Filter shirts by brand
- Hover animations and gradients
- Loading spinner while data loads
- Error handling

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- JSON Server
- Google Fonts
- Font Awesome
